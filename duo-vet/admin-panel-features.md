# Painel Admin - Funcionalidades sugeridas

## 1. Configuração de Login Social
- Cadastro de redirect URI
- Chaves OAuth (Google, Facebook, etc.)
- Ativar/desativar provedores

## 2. Stripe / Pagamentos
- Chave Stripe (public e secret)
- Configuração dos valores dos planos
- Visualizar e editar planos
- Renovar plano de usuário manualmente
- Histórico de pagamentos

## 3. Usuários
- Listar todos os usuários
- Editar dados e permissões
- Ativar/desativar usuários
- Resetar senha
- Filtrar por status/plano

## 4. Equipe / Permissões
- Gerenciar papéis (admin, veterinário, auxiliar)
- Convidar usuários
- Remover usuários
- Definir permissões customizadas

## 5. Auditoria / Logs
- Visualizar logs de acesso
- Histórico de ações administrativas

## 6. Configurações Gerais
- Variáveis de ambiente
- Dados da organização (nome, CNPJ, endereço)
- Personalização visual (logo, cores)

## 7. Utilidades
- Exportar dados (CSV, PDF)
- Backup manual
- Testar integração de APIs

---

Essas funções cobrem as necessidades de gestão, segurança, pagamentos e integrações. Recomendo priorizar login social, Stripe, usuários e equipe. Se quiser, posso iniciar a estrutura do painel admin com essas funcionalidades.
